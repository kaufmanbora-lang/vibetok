VibeTok: что загружать

1. Если нужен быстрый просмотр на GitHub Pages:
   Открой архив UPLOAD_THIS_TO_GITHUB_PAGES.zip и загрузи СОДЕРЖИМОЕ архива в репозиторий/Pages.
   Там должны быть index.html, 404.html, favicon.svg, .nojekyll и папка assets.

2. Если нужны аккаунты, которые видны всем пользователям, подписчики, лайки, админка и общая база:
   Загрузи VIBETOK_SHARED_FULLSTACK_SOURCE.zip в GitHub-репозиторий и подключи этот репозиторий к Render.
   GitHub Pages сам по себе не запускает сервер и не может хранить общие аккаунты для всех.

3. Если на GitHub Pages белый экран:
   Значит, скорее всего, загружены исходники проекта вместо готовой папки dist.
   Для GitHub Pages нужен именно UPLOAD_THIS_TO_GITHUB_PAGES.zip.
